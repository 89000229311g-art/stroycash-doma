# StroyCash Site

Это статический сайт‑строитель **StroyCash**.  
Он развернут на домене **doma.stroycash.ru** и обслуживается через Nginx.

## Структура проекта
```
├── index.html            # главная страница
├── css/                  # стили
├── js/                   # скрипты
├── images/               # изображения
└── *.html                # отдельные страницы
```

## Как запустить локально
```bash
# из корневой директории проекта
python3 -m http.server 8000   # откроет http://localhost:8000
```

## Деплой
Код хранится в репозитории **https://github.com/89000229311g-art/stroycash-site**.  
При каждом `push` в ветку `main` срабатывает GitHub Actions, который копирует файлы на ваш сервер и перезапускает Nginx.

## Домен
Сайт доступен по адресу: https://doma.stroycash.ru
